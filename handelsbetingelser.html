gavel 2. Anvendelsesområde

Disse handelsbetingelser gælder for alle aftaler mellem Nordic Autos ApS og kunder om:

  • Køb og salg af brugte biler
  • Import af biler fra udlandet
  • Konsulentydelser i forbindelse med bilkøb
  • Service og vedligeholdelse
  • Andre ydelser leveret af Nordic Autos

Vigtigt: Ved indgåelse af aftale accepterer kunden disse handelsbetingelser. Eventuelle afvigelser skal aftales skriftligt.

shopping_cart 3. Tilbud og bestilling

Tilbud

  • Alle priser på hjemmesiden er vejledende og kan ændres uden varsel
  • Tilbud er gyldige i 14 dage, medmindre andet er angivet
  • Priser er angivet i danske kroner inkl. moms og afgifter
  • Eventuelle finansieringsomkostninger kommer oveni

Bestilling og aftale

  • Aftale indgås ved underskrift af købekontrakt
  • Mundtlige aftaler skal bekræftes skriftligt
  • Nordic Autos forbeholder sig ret til at afvise ordrer
  • Ved import: Endelig pris først fastlåst ved kontraktunderskrift

Fortrydelsesret

Erhvervskunder: Ingen fortrydelsesret på bilkøb.
Privatkunder: 14 dages fortrydelsesret på konsulentydelser. Bilkøb er undtaget fortrydelsesret jf. forbrugeraftaleloven § 18, stk. 2, nr. 1.

payments 4. Priser og betaling

Priser

  • Alle priser er angivet i danske kroner inkl. moms
  • Priser inkluderer registreringsafgift og andre lovpligtige afgifter
  • Leveringsomkostninger tillægges efter aftale
  • Ved import: Eventuelle told- og afgiftsændringer påvirker slutprisen

Betaling

Accepterede betalingsmetoder

  • • Bankoverførsel
  • • Finansiering gennem samarbejdspartnere
  • • Byttehandel (efter vurdering)

Betalingsbetingelser

  • • Depositum: 10-50% ved kontraktindgåelse
  • • Restbeløb: Ved levering/afhentning
  • • Import: Fuld betaling før hjemtagelse
  • • Konsulentydelser: Forudbetaling

Forsinket betaling

Ved forsinket betaling påløber renter jf. renteloven samt et gebyr på 100 kr. Nordic Autos forbeholder sig ret til at tilbageholde bilen indtil fuld betaling.

local_shipping 5. Levering og afhentning

Leveringstid

  • Lagerbiler: Klar til afhentning efter fuld betaling og papirarbejde
  • Import fra Tyskland: 2-4 uger efter kontraktindgåelse
  • Specialordrer: Efter individuel aftale
  • Leveringstider er vejledende og kan påvirkes af eksterne faktorer

Levering og afhentning

Afhentning hos Nordic Autos

  • • Gratis afhentning på vores adresse
  • • Åbningstider efter aftale
  • • Gennemgang af bil ved afhentning
  • • Medbringe gyldigt kørekort og ID

Levering til kunde

  • • Levering efter aftale
  • • Leveringsomkostninger efter afstand
  • • Kun inden for Danmark
  • • Kunden skal være til stede ved levering

Forsinkelse

Ved forsinkelse ud over 30 dage kan kunden hæve købet og få depositum retur. Nordic Autos hæfter ikke for indirekte tab som følge af forsinkelse.

verified 6. Reklamation og garanti

Lovpligtig reklamationsret

  • Privatkunder: 24 måneders reklamationsret jf. købeloven
  • Erhvervskunder: Reklamation skal ske straks efter opdagelse
  • Reklamation skal fremsættes skriftligt med beskrivelse af fejlen
  • Bilen skal fremvises for vurdering hos Nordic Autos

Frivillig garanti

Nordic Autos Garanti

  • • Op til 24 måneders mekanisk garanti
  • • Dækker motor, gearkasse og andre væsentlige komponenter
  • • Gælder ved normal brug og vedligeholdelse
  • • Detaljerede garantibetingelser udleveres ved køb

Undtagelser fra garanti

  • Slitage og normale vedligeholdelsesarbejder
  • Skader som følge af uheld, misbrug eller manglende vedligeholdelse
  • Ændringer eller reparationer udført af andre end autoriserede værksteder
  • Kosmetiske fejl som ikke påvirker bilens funktion

shield 7. Ansvarsbegrænsning

Nordic Autos' ansvar er begrænset som følger:

  • Ansvar er begrænset til direkte tab og maksimalt købesummen
  • Intet ansvar for indirekte tab som driftstab, tabt fortjeneste mv.
  • Ansvar forudsætter, at kunden har overholdt sine forpligtelser
  • Force majeure fritager Nordic Autos for ansvar

Vigtigt: Ansvarsbegrænsningen gælder ikke ved grov uagtsomhed eller forsæt fra Nordic Autos' side.

import_export 8. Import og konsulentydelser

Konsulentpakke

  • Pris fra 15.000 kr. + moms for komplet importservice
  • Inkluderer søgning, forhandling, transport og registrering
  • Kunden betaler bilens pris + importomkostninger + konsulenthonorar
  • Endelig pris afhænger af bilens værdi og kompleksitet

Kundens forpligtelser ved import

  • Forudbetaling af konsulenthonorar
  • Godkendelse af bil før køb i udlandet
  • Betaling af bilens pris + omkostninger før hjemtagelse
  • Tålmodighed med leveringstid (2-4 uger)

Særlige forhold ved import

  • • Priser kan ændres pga. valutaudsving og afgiftsændringer
  • • Nordic Autos kan ikke garantere for skjulte fejl ved udenlandske biler
  • • Kunden accepterer bilens tilstand som beskrevet
  • • Eventuelle reklamationer håndteres efter dansk ret

privacy_tip 9. Behandling af personoplysninger

Nordic Autos behandler personoplysninger i overensstemmelse med GDPR og dansk lovgivning.

Formål med databehandling

  • • Opfyldelse af købekontrakt og levering
  • • Kundeservice og support
  • • Lovpligtig bogføring og dokumentation
  • • Markedsføring (kun med samtykke)

Læs mere i vores privatlivspolitik.

update 11. Ændringer i handelsbetingelser

Nordic Autos kan ændre disse handelsbetingelser med 30 dages varsel.

  • Ændringer meddeles på hjemmesiden og via email til eksisterende kunder
  • Væsentlige ændringer kræver kundens accept
  • Eksisterende aftaler påvirkes ikke af ændringer
  • Kunden kan opsige aftaler ved uacceptable ændringer

Anbefaling: Gennemgå handelsbetingelserne regelmæssigt for at holde dig opdateret.
